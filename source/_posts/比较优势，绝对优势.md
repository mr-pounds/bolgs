**比较优势是指谁付出的机会成本比较低；绝对优势是指谁付出的直接成本比较低。**

A、B 都能生产鱼和玉米，且 A 的生产效率均比 B 高。

| 类目 | 鱼        | 玉米       |
| ---- | --------- | ---------- |
| A    | 5 条/小时 | 50 斤/小时 |
| B    | 3 条/小时 | 40 斤/小时 |

如果按绝对优势比较，A、B 各自生产 (每天 8 小时)，但实际上会导致整体的生产效率降低。

| 类目 | 鱼    | 玉米   |
| ---- | ----- | ------ |
| A    | 20 条 | 200 斤 |
| B    | 12 条 | 160 斤 |
| 总计 | 32 条 | 360 斤 |

如果按比较优势，则机会成本如下

| 类目 | 鱼                 | 玉米                |
| ---- | ------------------ | ------------------- |
| A    | 10 斤玉米/1 条鱼   | 0.1 条鱼/1 斤玉米   |
| B    | 13.3 斤玉米/1 条鱼 | 0.075 条鱼/1 斤玉米 |

生产鱼来说，B 的机会成本更大；生产玉米来说，A 的机会成本更大。则 A 负责生产鱼，B 负责生产玉米

| 类目 | 鱼    | 玉米   |
| ---- | ----- | ------ |
| A    | 40 条 | 0 斤   |
| B    | 0 条  | 320 斤 |
| 总计 | 40 条 | 320 斤 |

乍一看，鱼生产多了，但是玉米生产少了。但是如果按市场价值来说，则不一定。因为鱼的产量低，市场价值更大。假设 8 条鱼=90 斤玉米（因为当前市场只有这些商品和产量），那么多生产 8 条鱼，相当于多生产 90 斤玉米。因此按比较优势的方案，产生了贸易和分工，但是市场变大了。

**这是一个反直觉的观点，所以特地按自己的思路梳理了一遍**。
